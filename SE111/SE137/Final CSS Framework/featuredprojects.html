Software Layout

05/27/24


This project was assigned during Week 6 and included many features that were required by incorporating HTML, CSS and Javascript through external sheets. The goal of this project was to practice expanding boxes, practice hover effects, such as when you hover over the icons. The side tray part of the code editor needed an action that allowed it to close and open through Javascript functions.

This project was made to simulate a code editor such a Visual studioc code for example; utilizing the concept of grids, columns and rows was frustrating initally, but as always FLEXBOX!

Although I didn't recieve the grade I wished for, I was pleased with the outcome, because of the hardwork and learning curve it was for me. Javascript and getting the items aligned were the most challeging aspects of this project.
